cd - change dir смена директории
ls - отображение списка файлов и директорий в текущей директории
mv - переименование файла либо его перезапись в другую директорию
mkdir - создание директории вторым аргументом передается название директории
sudo - запуск комманды от root прав
apt - пакетный менеджер для установки пакетов
cat - вывод данных из файла
tail - отображение конца файла
head - отображение начала файла
touch создание файла
ssh - подключение к удаленному серверу по ssh 
scp - отправка с файлов с сервера на клиент/с клиента на сервер по ssh протоколу
history - отображение истории вводимых команд
nano - редактор текста
htop - отображение запущенных процессов
unzip - распаковка zip архивов
cmake - сборщик
make - сборщик
gcc - компилятор С/C++
git clone - клонирование репозитория с git
